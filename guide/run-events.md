{.is-info}

## Runs & Events 
- **Filters**: Upcoming, Past, All.
- **Columns**: Run #, Date, Hare, Run Site, Status, Actions (edit/delete).
- **Quick actions**: Add Event, Schedule Run.
- **Status badges**: Scheduled, Published, or Run Site Needed to highlight gaps.

![runs.png](/guide/runs.png)
*(image: runs list with filters Upcoming/Past/All)*

- **Create a run**: Set date/time, start location, hares, and notes.
- **Media & routes**: Attach GPX files, maps, and photos.
- **Status**: Draft, published, completed, or canceled; keep the hareline updated.

- **Edits**: Update details; notify members as needed.

## Creating or editing a run
- **General Details**: Run number, date, run site, hare, run name/group.
- **Report & Notes**: Run report and internal notes.
- **Media & Files**: GPX/routes, maps, and photos of the trail.
- **Stats & Data**: Attendees, distance (km), elevation (m).

## Events
- **Create or edit an event**: Title, status, start/end dates, registration deadline, capacity, and homepage promotion.
- **Tabs**: General, Content, Venue & Schedule, Pricing & Add-ons, Registration, Contact Info, Settings.
- **Status**: Draft/Published with date-based visibility.
