> Plan and track your runs, keep the hareline filled, and record updates on each event.
{.is-info}

- **Create a run**: Set date/time, start location, hares, and notes.
- **Media & routes**: Attach GPX files, maps, and photos.
- **Status**: Draft, published, completed, or canceled; keep the hareline updated.
- **Events stream**: View updates and actions related to runs.
- **Edits**: Update details; notify members as needed.

## Runs list
- **Tabs**: Upcoming, Past, All.
- **Columns**: Run #, Date, Hare, Run Site, Status, Actions (edit/delete).
- **Quick actions**: Add Event, Schedule Run.
- **Status badges**: Scheduled, Published, or Run Site Needed to highlight gaps.

*(screenshot: runs list with tabs Upcoming/Past/All and status badges)*

## Creating or editing a run
- **Basics**: Title, date/time, start location (choose from run sites), hares, notes.
- **Attachments**: GPX/routes, maps, and photos for the trail.
- **Visibility**: Save as draft or publish when ready.
- **Updates**: Edit details; changes show in the events stream.

*(screenshot: run create/edit form with date, location, hares, status)*
*(screenshot: run detail page with attachments and status)*

## Hareline status
- See how many weeks out your runs are scheduled.
- Fill gaps by adding new runs; link to run sites to speed entry.

*(screenshot: hareline status widget showing weeks covered and upcoming runs)*

## Edit tabs
- **General Details**: Run number, date, run site, hare, run name/group.
- **Report & Notes**: Run report and internal notes.
- **Media & Files**: Upload images and GPX.
- **Stats & Data**: Attendees, distance (km), elevation (m).
